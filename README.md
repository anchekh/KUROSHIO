```markdown
# Установка проекта

1. Клонирование репозитория
# git clone https://github.com/anchekh/KUROSHIO.git

2. Установка зависимостей
# npm install

3. Запуск сервера разработки
# npm run dev ( доступная ссылка - http://localhost:5173 )

4. Сборка для production
# npm run build

5. Превью сборки
# npm run preview
```
